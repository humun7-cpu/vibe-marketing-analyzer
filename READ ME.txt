실행 방법

1. 프로젝트 폴더를 연다.
2. 터미널에서 아래 명령어 실행:
   npm install
3. 루트 폴더에 .env.local 파일 생성 후 아래 입력:
   GEMINI_API_KEY=본인 Gemini API 키
4. 아래 명령어 실행:
   npm run dev
5. 브라우저에서 http://localhost:3000 접속

참고:
- PowerShell에서 npm 실행이 안 되면 npm.cmd install, npm.cmd run dev 사용
- 로그인/권한이 필요한 URL은 분석이 제한될 수 있음
